## 产品简介

TG-Matrix 是一款专业的 Telegram 营销自动化工具，专为营销人员、销售团队和社群运营者设计。它能帮助您：

- 🎯 **精准获客** - 通过关键词监控自动捕获潜在客户
- 🤖 **AI 智能回复** - 接入 Gemini、OpenAI、本地模型，实现智能自动回复
- 📊 **多账号管理** - 统一管理多个 Telegram 账号
- 🔄 **自动化流程** - 设置自动化营销工作流
- 📈 **数据分析** - 追踪业绩和转化率

## 快速开始

### 系统要求

- **操作系统**: Windows 10/11 (64位)
- **内存**: 最低 4GB RAM（推荐 8GB）
- **存储**: 500MB 可用磁盘空间
- **网络**: 稳定的互联网连接

### 安装步骤

1. 从官方网站下载最新版本
2. 运行安装程序并按提示操作
3. 启动 TG-Matrix
4. 完成初始设置向导

### 首次设置

1. **添加第一个账号**
   - 点击侧边栏的"添加账号"
   - 输入 Telegram 手机号
   - 完成手机验证
   - 或导入现有的 Session 文件

2. **配置关键词**
   - 进入"监控"页面
   - 添加要追踪的关键词
   - 选择要监控的群组

3. **设置 AI 助手**
   - 进入"AI 中心"
   - 输入 API 密钥
   - 自定义 AI 人设

## 账号管理

### 添加账号

#### 方式一：手机号登录
1. 点击"添加账号"
2. 输入带国家代码的手机号
3. 输入验证码
4. 可选：输入两步验证密码

#### 方式二：导入 Session
1. 点击"导入 Session"
2. 选择 `.session` 或 `.tgpkg` 文件
3. 如提示则输入 API 凭据

### 账号状态

| 状态 | 说明 |
|------|------|
| 🟢 已连接 | 账号活跃，可正常使用 |
| 🟡 连接中 | 正在建立连接 |
| 🔴 已断开 | 连接丢失 |
| ⚠️ 受限 | 账号存在限制 |

### 代理配置

为提高稳定性，建议配置代理：

1. 点击账号设置（⚙️）
2. 选择"代理设置"
3. 选择代理类型（SOCKS5/HTTP）
4. 输入代理详情

## 关键词监控

### 创建关键词组

1. 进入"监控"页面
2. 点击"添加关键词组"
3. 输入关键词（每行一个）
4. 启用选项：
   - 区分大小写
   - 正则表达式支持
   - 排除模式

### 监控群组

1. 点击"添加群组"
2. 输入群组用户名或链接
3. 选择监控关键词
4. 设置提醒偏好

### 捕获的线索

当检测到关键词时，系统自动：
- 添加到线索列表
- 按来源分类
- 准备好进行跟进

## AI 助手

### 支持的提供商

| 提供商 | 特点 |
|--------|------|
| Google Gemini | 快速、可靠、多语言 |
| OpenAI GPT | 强大、灵活 |
| 本地模型 | 注重隐私、离线可用 |

### 配置步骤

1. 选择 AI 提供商
2. 输入 API 密钥
3. 选择模型版本
4. 设置参数：
   - 温度（创造性）
   - 最大 Token（回复长度）

### AI 人设

自定义 AI 回复风格：

```
你是 [公司] 的友好销售代表。
你的目标是了解客户需求并提供有用的信息。
始终保持专业和礼貌。
```

### 知识库 (RAG)

上传文档以增强 AI 回复：
- 产品目录
- FAQ 文档
- 定价信息
- 公司政策

## 自动化营销

### 创建活动

1. 点击"创建活动"
2. 设置触发条件：
   - 捕获新线索
   - 关键词匹配
   - 定时触发
3. 定义操作：
   - 发送消息
   - 分配给团队成员
   - 添加标签
4. 激活活动

### 工作流示例

**欢迎序列**
```
触发：新线索
→ 等待 5 分钟
→ 发送欢迎消息
→ 等待 24 小时
→ 如无回复则发送跟进消息
```

**关键词响应**
```
触发：关键词"价格"
→ 生成包含定价信息的 AI 回复
→ 发送给线索
→ 标记为"有意向"
```

## 常见问题

### 通用问题

**问：TG-Matrix 安全吗？**
答：是的。我们使用安全的会话管理和加密存储。您的凭据永远不会传输到我们的服务器。

**问：可以使用多个账号吗？**
答：可以，根据您的会员方案而定。免费用户可使用 1 个账号，VIP 用户最多 3 个，以此类推。

**问：支持 Mac/Linux 吗？**
答：目前仅支持 Windows。Mac 和 Linux 版本正在开发中。

### 技术问题

**问：为什么账号断开了？**
答：常见原因：
- 网络问题
- Session 过期
- Telegram 限制账号

**问：如何备份数据？**
答：进入 设置 → 备份 → 创建备份。您也可以启用自动备份。

**问：可以导出线索吗？**
答：可以。进入 潜在客户 → 导出 → 选择格式（CSV/JSON）。

### 付款问题

**问：接受哪些付款方式？**
答：我们接受支付宝、微信支付、信用卡（Stripe）和 USDT。

**问：有退款政策吗？**
答：有。购买后 7 天内联系客服可全额退款。
