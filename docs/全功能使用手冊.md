## 快速入門

### 🚀 第一次使用

1. **安裝應用程式**
   - 雙擊安裝包 `TG-AI智控王-Setup.exe`
   - 選擇安裝路徑，點擊「安裝」
   - 等待安裝完成，點擊「完成」啟動

2. **添加 Telegram 帳號**
   - 進入「帳號管理」頁面
   - 點擊「+ 添加帳號」
   - 選擇登入方式（QR碼/手機號/Session文件）
   - 完成驗證

3. **開始使用**
   - 帳號登入成功後，即可使用各項功能
   - 建議先配置「自動化規則」實現自動回覆

### 🎯 核心功能概覽

| 模塊 | 功能 | 說明 |
|------|------|------|
| 📊 儀表板 | 系統總覽 | 快速查看所有關鍵數據 |

| ⚡ 自動化 | 自動回覆 | 關鍵詞觸發自動回覆 |
| 📋 客戶管理 | 線索培育 | 管理潛在客戶 |

| 🧠 AI 中心 | 智能對話 | AI 驅動的對話引擎 |
| 🔭 資源發現 | 群組搜索 | 發現優質群組資源 |

## 儀表板

### 📊 功能說明

儀表板是應用的首頁，提供系統運行狀態的整體概覽。

### 主要區域

1. **帳號狀態卡片**
   - 顯示所有帳號的在線狀態
   - 綠色 = 在線，紅色 = 離線，黃色 = 連接中

2. **今日統計**
   - 發送消息數
   - 接收消息數
   - AI 回覆次數
   - 新增線索數

3. **快捷操作**
   - 一鍵啟動監控
   - 快速發送消息
   - 查看最新告警

4. **最近活動**
   - 顯示最近的系統活動記錄
   - 包括發送、接收、觸發等事件

### 使用技巧

- 按 `D` 鍵可快速返回儀表板
- 點擊任意統計卡片可進入詳細頁面
- 右上角顯示系統資源使用情況

## 帳號管理

### 👤 功能說明

管理所有 Telegram 帳號，包括添加、刪除、配置帳號。

### 子功能

#### 1. 帳號列表

顯示所有已添加的 Telegram 帳號。

**操作說明：**
- **查看帳號信息**：點擊帳號卡片展開詳情
- **刷新狀態**：點擊刷新按鈕更新帳號狀態
- **刪除帳號**：右鍵菜單或點擊刪除按鈕
- **批量操作**：勾選多個帳號進行批量操作

**帳號狀態說明：**
| 狀態 | 顏色 | 說明 |
|------|------|------|
| 在線 | 🟢 綠色 | 帳號已連接可用 |
| 離線 | 🔴 紅色 | 帳號未連接 |
| 連接中 | 🟡 黃色 | 正在嘗試連接 |
| 受限 | 🟠 橙色 | 帳號被限制 |
| 封禁 | ⚫ 黑色 | 帳號已被封禁 |

#### 2. 添加帳號

**三種登入方式：**

**方式一：QR碼登入（推薦）**
1. 點擊「QR碼登入」
2. 使用 Telegram App 掃描二維碼
3. 在手機上確認登入
4. 等待驗證完成

**方式二：手機號登入**
1. 輸入手機號（含國家代碼，如 +86）
2. 點擊「發送驗證碼」
3. 輸入收到的驗證碼
4. 如有二次驗證密碼，輸入密碼

**方式三：Session 文件導入**
1. 點擊「導入 Session」
2. 選擇 `.session` 文件
3. 系統自動驗證並添加

#### 3. API 憑證管理

管理 Telegram API ID 和 Hash。

**獲取 API 憑證：**
1. 訪問 https://my.telegram.org
2. 登入您的 Telegram 帳號
3. 進入「API Development Tools」
4. 創建應用獲取 API ID 和 Hash

**配置步驟：**
1. 點擊「添加 API 憑證」
2. 輸入 API ID 和 API Hash
3. 為憑證命名（便於識別）
4. 點擊保存

#### 4. 帳號健康監控

監控帳號的健康狀態，預防封號風險。

**監控指標：**
- 發送頻率
- 被舉報次數
- 驗證碼請求次數
- 異常行為檢測

## 自動化中心

### ⚡ 功能說明

設置自動化規則，實現關鍵詞觸發自動回覆、自動收集用戶等功能。

### 子功能

#### 1. 自動化總覽

顯示所有自動化規則的運行狀態。

**主要操作：**
- 一鍵啟動/停止所有監控
- 查看規則觸發統計
- 快速創建新規則

#### 2. 監控群組

選擇需要監控的群組。

**操作步驟：**
1. 點擊「添加監控群組」
2. 選擇帳號
3. 從群組列表中選擇要監控的群組
4. 設置監控選項：
   - ✅ 監控新消息
   - ✅ 監控新成員
   - ✅ 自動收集用戶信息

**批量添加：**
1. 點擊「批量添加」
2. 粘貼群組鏈接（每行一個）
3. 選擇執行帳號
4. 點擊「開始加入」

#### 3. 關鍵詞集

配置觸發關鍵詞。

**創建關鍵詞集：**
1. 點擊「新建關鍵詞集」
2. 輸入名稱（如：「購買意向」）
3. 添加關鍵詞：
   - 支持精確匹配：`購買`
   - 支持模糊匹配：`*買*`
   - 支持正則表達式：`/買|購|訂/`
4. 設置優先級
5. 保存

**關鍵詞示例：**
```
# 購買意向
購買, 怎麼買, 價格, 多少錢, 哪裡買

# 諮詢類
請問, 想了解, 有沒有, 能不能

# 排除詞（不觸發）
廣告, 推銷, 群發
```

#### 4. 觸發規則

配置觸發條件和執行動作。

**創建觸發規則：**
1. 點擊「新建規則」
2. 設置觸發條件：
   - 匹配關鍵詞集
   - 消息來源（私聊/群組）
   - 用戶類型（新用戶/老用戶）
   - 時間條件
3. 設置執行動作：
   - 發送回覆消息
   - 添加到線索庫
   - 打標籤
   - 發送通知
4. 設置執行參數：
   - 延遲時間（模擬人工）
   - 每日最大觸發次數
5. 保存並啟用

#### 5. 聊天模板

預設回覆模板。

**創建模板：**
1. 點擊「新建模板」
2. 輸入模板名稱
3. 編輯模板內容，支持變量：
   - `{name}` - 用戶名
   - `{time}` - 當前時間
   - `{random}` - 隨機文本
4. 設置模板類型（問候/回覆/跟進）
5. 保存

**模板示例：**
```
您好 {name}，感謝您的關注！

我是 XXX 的客服，請問有什麼可以幫助您的嗎？

回覆「1」了解產品
回覆「2」查看價格
回覆「3」聯繫客服
```

## 客戶管理

### 📋 功能說明

管理潛在客戶（線索），進行客戶培育和跟進。

### 子功能

#### 1. 線索管理

管理所有潛在客戶。

**線索來源：**
- 自動收集（監控群組）
- 手動添加
- 批量導入

**線索狀態：**
| 狀態 | 說明 |
|------|------|
| 🆕 新線索 | 剛收集，未聯繫 |
| 📞 已聯繫 | 已發送首次消息 |
| 💬 對話中 | 正在溝通 |
| 🔥 高意向 | 購買意向明確 |
| ✅ 已成交 | 完成轉化 |
| ❌ 無意向 | 明確拒絕 |

**操作說明：**
- **篩選線索**：按狀態、標籤、來源篩選
- **批量操作**：批量打標籤、批量發送
- **導出線索**：導出為 Excel/CSV

#### 2. 線索培育

AI 驅動的自動培育流程。

**培育流程設置：**
1. 創建培育計劃
2. 設置培育階段：
   - 第1天：發送歡迎消息
   - 第3天：發送產品介紹
   - 第7天：發送優惠信息
   - 第14天：發送跟進消息
3. 啟動自動執行

**智能培育特性：**
- 根據用戶回覆自動調整節奏
- AI 分析用戶意向
- 自動識別最佳聯繫時間

#### 3. 成員數據庫

從群組提取的成員數據。

**提取成員：**
1. 選擇目標群組
2. 點擊「提取成員」
3. 設置過濾條件：
   - 最近活躍時間
   - 排除機器人
   - 排除管理員
4. 開始提取

**數據字段：**
- 用戶 ID
- 用戶名
- 顯示名稱
- 最後活躍時間
- 所屬群組

#### 4. 用戶追蹤

追蹤高價值用戶的動態。

**追蹤功能：**
- 在線狀態監控
- 群組活動追蹤
- 消息記錄

## 營銷中心

### 📢 功能說明

批量發送廣告消息，管理營銷活動。

### 子功能

#### 1. 廣告發送

批量發送廣告消息到群組或私聊。

**創建發送任務：**
1. 選擇發送帳號
2. 選擇目標（群組/用戶）
3. 編輯消息內容
4. 設置發送參數：
   - 發送間隔（建議 30-60 秒）
   - 每帳號每日限額
   - 隨機延遲
5. 預覽並發送

**安全設置：**
- ✅ 啟用隨機間隔（防封必選）
- ✅ 模擬打字效果
- ✅ 避開高峰時段
- ✅ 失敗自動重試

#### 2. 營銷活動

創建和管理營銷活動。

**活動類型：**
- 新品推廣
- 限時優惠
- 節日促銷
- 常規推廣

**創建活動：**
1. 設置活動名稱和時間
2. 選擇目標受眾
3. 設計消息內容
4. 設置發送計劃
5. 啟動活動

#### 3. 多角色協作

多個 AI 角色配合進行群組營銷。

**使用場景：**
在群組中，多個帳號扮演不同角色（客戶、客服、專家），通過對話吸引真實用戶。

**設置步驟：**

**步驟一：創建角色**
1. 進入「多角色協作」
2. 點擊「添加角色」
3. 設置角色信息：
   - 角色名稱（如：產品體驗官）
   - 角色描述
   - 說話風格
   - 關聯帳號
4. 保存角色

**步驟二：創建劇本**
1. 點擊「創建劇本」
2. 編寫對話劇本：
```
[體驗官] 最近用了 XX 產品，效果真的很好！
[圍觀者] 真的嗎？效果怎麼樣？
[體驗官] 用了一週，明顯感覺...
[專家] 這個產品的核心成分是...
```
3. 設置執行時間和間隔
4. 保存劇本

**步驟三：執行協作**
1. 選擇目標群組
2. 選擇執行帳號
3. 選擇劇本
4. 開始執行

**AI 自動模式：**
- 系統自動生成對話內容
- 根據群組氛圍調整節奏
- 智能識別介入時機

## AI 中心

### 🧠 功能說明

AI 驅動的智能對話引擎，提供自動回覆和內容生成。

### 子功能

#### 1. AI 對話引擎

配置 AI 自動回覆。

**配置 AI 模型：**
1. 選擇 AI 提供商：
   - OpenAI (GPT-4/GPT-3.5)
   - Google Gemini
   - Ollama (本地模型)
   - DeepSeek
2. 輸入 API Key
3. 測試連接

**設置 AI 回覆：**
1. 創建 AI 回覆規則
2. 設置觸發條件
3. 編寫系統提示詞（Prompt）：
```
你是一個專業的產品顧問，請根據用戶問題提供專業、友好的回答。

注意事項：
- 保持專業但親切的語氣
- 回答控制在 50-100 字
- 適時引導用戶了解產品
- 不要主動提及競品
```
4. 設置回覆參數
5. 啟用規則

**知識庫配置：**
1. 點擊「知識庫」
2. 上傳文檔（PDF/TXT/Markdown）
3. 系統自動索引
4. AI 回覆時自動檢索相關知識

#### 2. AI 營銷助手

AI 輔助生成營銷內容。

**功能：**
- 生成廣告文案
- 優化消息內容
- 生成回覆建議
- 分析對話意向

**使用方式：**
1. 輸入原始內容或需求
2. 選擇生成類型
3. 點擊「AI 生成」
4. 編輯優化
5. 使用內容

## 常見問題

### Q1: 帳號無法登入怎麼辦？

**可能原因：**
- 網絡連接問題
- API 憑證錯誤
- 帳號被限制

**解決方法：**
1. 檢查網絡連接
2. 確認 API ID 和 Hash 正確
3. 嘗試使用代理
4. 等待一段時間後重試

### Q2: 自動回覆沒有觸發？

**檢查項目：**
1. 監控是否已啟動（綠色狀態）
2. 群組是否在監控列表中
3. 關鍵詞是否匹配
4. 觸發規則是否啟用
5. 帳號是否在線

### Q3: 發送消息失敗？

**可能原因：**
- 帳號被限流
- 發送間隔太短
- 目標不可達

**解決方法：**
1. 增加發送間隔（至少 30 秒）
2. 減少每日發送量
3. 檢查目標是否有效
4. 使用多個帳號輪換

### Q4: AI 回覆質量不好？

**優化建議：**
1. 優化系統提示詞
2. 上傳更多知識庫文檔
3. 調整回覆長度參數
4. 選擇更高級的模型

### Q5: 如何避免帳號被封？

**防封建議：**
1. ✅ 使用隨機發送間隔
2. ✅ 控制每日發送量
3. ✅ 避免發送敏感內容
4. ✅ 使用代理分散 IP
5. ✅ 帳號養號一段時間再用
6. ❌ 不要短時間大量發送
7. ❌ 不要發送明顯廣告內容

### Q6: 數據如何備份？

1. 進入「設置」→「數據管理」
2. 點擊「創建備份」
3. 選擇備份內容
4. 等待備份完成
5. 下載備份文件
