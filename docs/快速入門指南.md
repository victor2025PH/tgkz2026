# TG-AI智控王 快速入門指南

> 5 分鐘快速上手

---

## 🚀 第一步：安裝啟動

```
雙擊安裝包 → 點擊「安裝」 → 完成後啟動應用
```

---

## 👤 第二步：添加帳號

### 方式一：QR碼登入（推薦）

```
帳號管理 → 添加帳號 → QR碼登入 → 手機掃碼確認
```

### 方式二：手機號登入

```
帳號管理 → 添加帳號 → 手機號登入 → 輸入驗證碼
```

---

## ⚡ 第三步：設置自動回覆

### 3.1 創建關鍵詞

```
自動化 → 關鍵詞集 → 新建 → 輸入關鍵詞（如：購買,價格,怎麼買）
```

### 3.2 創建回覆模板

```
自動化 → 聊天模板 → 新建 → 編輯回覆內容
```

### 3.3 創建觸發規則

```
自動化 → 觸發規則 → 新建規則 → 選擇關鍵詞 → 選擇回覆模板 → 保存
```

---

## 👥 第四步：添加監控群組

```
自動化 → 監控群組 → 添加群組 → 選擇帳號 → 選擇群組 → 確認
```

---

## ▶️ 第五步：啟動監控

```
儀表板 → 點擊「啟動監控」按鈕 → 狀態變為綠色
```

---

## ✅ 完成！

現在系統會自動：
- ✅ 監控群組消息
- ✅ 匹配關鍵詞觸發
- ✅ 自動發送回覆
- ✅ 收集用戶線索

## 📊 核心功能速覽

| 我想要... | 去哪裡 | 怎麼做 |
|-----------|--------|--------|

## ⚠️ 重要提示

### 防封建議

| ✅ 應該做 | ❌ 不要做 |
|----------|----------|
| 發送間隔 30-60 秒 | 連續快速發送 |
| 每天限制 50-100 條 | 一天發幾百條 |
| 使用自然語言 | 明顯的廣告模板 |
| 帳號養號再用 | 新號立即大量操作 |
