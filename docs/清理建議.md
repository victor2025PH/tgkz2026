## 3. 其他清理建議

### 3.1 dist 目錄
`dist/` 目錄包含構建輸出，應該加入 `.gitignore`（如果還沒有）。

### 3.2 測試結果目錄
`test-results/` 和 `playwright-report/` 應該加入 `.gitignore`。

### 3.3 臨時文本文件
- `npm run startbuild.txt`
- `wushizhifu.txt`
- `新建文本文档.txt`
- `开发用户提示词.txt`

這些可能是臨時筆記，建議移到 `docs/notes/` 或刪除。

## 清理命令

如果確認要清理，可以使用以下命令：

### 創建歸檔目錄
```bash
mkdir archive
mkdir archive/fix_scripts
```

### 移動修復腳本到歸檔
```bash
mv fix_*.py archive/fix_scripts/
mv final_*.py archive/fix_scripts/
mv verify_fix.py archive/fix_scripts/
mv force_*.py archive/fix_scripts/
mv wait_and_restart.py archive/fix_scripts/
mv update_and_restart.py archive/fix_scripts/
mv ssh_fix_nginx.py archive/fix_scripts/
mv apply_ssl.py archive/fix_scripts/
mv setup_ssl_complete.py archive/fix_scripts/
```

### 確保 .gitignore 包含
```
# Build outputs
dist/
build/

# Test results
test-results/
playwright-report/

# Archive
archive/

# Temporary files
*.tmp
*.log
```
