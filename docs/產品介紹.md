### 2️⃣ 多角色協作引擎

**模擬真實銷售團隊，多角色配合提升轉化**

| 角色 | 職責 | 話術風格 |
|------|------|---------|
| 💼 成交專家 | 促成最終交易 | 專業、果斷 |
| 🎓 產品專家 | 解答技術問題 | 專業、詳細 |
| 🤝 客服支持 | 處理疑慮異議 | 耐心、同理 |
| ⭐ 見證者 | 分享使用經驗 | 真實、熱情 |
| 🌟 社交達人 | 建立信任關係 | 親切、友好 |

**協作模式**：
```
私聊協作：多個角色輪流與客戶私聊，從不同角度建立信任
     ↓
興趣識別：AI 實時分析客戶意向，識別購買信號
     ↓
自動建群：檢測到高意向後自動創建 VIP 群
     ↓
組群營銷：角色們在群內協作，營造購買氛圍
     ↓
促成交易：最終由成交專家完成轉化
```
