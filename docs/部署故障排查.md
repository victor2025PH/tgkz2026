# 部署故障排查

## Docker Compose: "no configuration file provided: not found"

**原因**：在錯誤的目錄執行 `docker compose`，找不到 `docker-compose.yml`。

**解決**：先進入項目部署目錄，再執行 compose 命令：

```bash
ssh ubuntu@165.154.210.154
cd /opt/tg-matrix
sudo docker compose logs -f
# 或
sudo docker compose ps
sudo docker compose up -d
```

## 管理後台優惠券彈窗無法關閉

若登錄後台仍彈出創建優惠券且按鈕無效：

1. **強制刷新**：按 `Ctrl+Shift+R` 或 `Cmd+Shift+R` 清除緩存
2. **Esc 關閉**：彈窗打開時按鍵盤 `Esc` 可關閉
3. **確認部署**：在服務器上檢查 `/opt/tg-matrix/admin-panel/` 下 `app.js` 和 `index.html` 是否為最新版本
