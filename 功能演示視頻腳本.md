## 視頻 1：軟體總覽與快速開始

**時長**：3-5 分鐘  
**目標**：讓新用戶快速了解軟體並完成首次登入

### 腳本

```
🎬 開場（15秒）
├── 展示軟體啟動畫面
├── 旁白：「歡迎使用 TG-AI智控王，這是一款專業的 Telegram 智能營銷系統」
└── 簡述核心功能：多帳號管理、智能監控、AI 營銷

🎬 界面導覽（60秒）
├── 展示側邊導航欄
├── 逐一介紹各功能區塊：
│   ├── 會員中心
│   ├── 儀表盤
│   ├── 手動操作區（帳號管理、搜索發現、資源中心、發送控制台）
│   ├── 監控中心區
│   └── AI 智能營銷區
└── 展示頂部狀態欄

🎬 添加帳號演示（90秒）
├── 點擊「帳號管理」
├── 點擊「添加帳號」按鈕
├── 展示兩種登入方式：
│   ├── 方式1：掃碼登入（推薦）
│   │   ├── 選擇 API 憑據
│   │   ├── 生成二維碼
│   │   └── 使用手機掃碼完成
│   └── 方式2：手機號登入
│       ├── 輸入手機號
│       ├── 接收驗證碼
│       └── 完成登入
└── 展示登入成功後的帳號卡片

🎬 儀表盤說明（45秒）
├── 切換到儀表盤
├── 說明各統計卡片含義
├── 展示系統狀態指示
└── 演示「系統診斷」功能

🎬 結尾（15秒）
├── 總結：「您已成功添加帳號，現在可以開始探索更多功能」
└── 引導觀看其他教學視頻
```

## 視頻 2：搜索發現與資源管理

**時長**：4-6 分鐘  
**目標**：演示如何搜索群組並管理資源

### 腳本

```
🎬 開場（10秒）
└── 「本視頻將演示如何使用搜索發現功能找到目標群組」

🎬 搜索群組（90秒）
├── 點擊「搜索發現」
├── 在搜索框輸入關鍵詞（如：「區塊鏈」「交流群」）
├── 選擇搜索帳號
├── 點擊搜索按鈕
├── 展示搜索結果列表
├── 說明各欄位含義：
│   ├── 群組名稱
│   ├── 成員數量
│   ├── 群組類型
│   └── 加入狀態
└── 演示結果篩選與排序

🎬 批量加群（60秒）
├── 勾選目標群組
├── 點擊「批量加入」
├── 展示加群進度
├── 說明速率控制（防封保護）
└── 展示加入結果

🎬 資源中心管理（90秒）
├── 切換到「資源中心」
├── 展示已加入的群組列表
├── 演示群組操作：
│   ├── 查看群組詳情
│   ├── 提取群成員
│   └── 群組分類標籤
├── 展示成員資源列表
└── 演示批量操作功能

🎬 成員提取演示（60秒）
├── 選擇一個群組
├── 點擊「提取成員」
├── 配置提取選項：
│   ├── 過濾在線狀態
│   ├── 排除 Bot
│   └── 數量限制
├── 開始提取
└── 展示提取結果

🎬 結尾（10秒）
└── 「資源已準備就緒，您可以開始進行營銷活動」
```

## 視頻錄製建議

### 準備工作
1. 確保軟體運行正常
2. 準備測試帳號（至少 2-3 個）
3. 清理敏感信息
4. 調整屏幕分辨率（建議 1920x1080）

### 錄製技巧
1. 使用清晰的滑鼠軌跡
2. 操作速度適中，不要太快
3. 重要操作後稍作停頓
4. 保持界面簡潔

### 後期處理
1. 添加字幕（建議）
2. 關鍵步驟添加箭頭/高亮
3. 背景音樂（輕柔不搶眼）
4. 片頭片尾品牌標識

### 建議視頻格式
- 分辨率：1920x1080 (1080p)
- 幀率：30fps
- 格式：MP4 (H.264)
- 音頻：AAC 128kbps
