## 🎯 使用流程

### 1. 启动应用
```powershell
npm start
```

### 2. 添加账户
- 手动添加
- Excel 导入
- Session 导入

### 3. 登录账户
- 输入验证码
- 输入 2FA（如果需要）

### 4. 配置监控
- 添加监控群组
- 添加关键词集
- 创建消息模板
- 创建自动化活动

### 5. 启动监控
- 点击"启动监控"
- 系统自动捕获潜在客户
- 活动自动触发并发送消息
