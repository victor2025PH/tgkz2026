# 立即安装 TgCrypto - 简单步骤

## ✅ 您已经安装了 C++ 工具，现在只需要：

### 方法 1：使用开发者命令提示符（最简单，推荐）⭐

1. **按 `Win + S`**，搜索 **"Developer Command Prompt"**

2. **打开 "Developer Command Prompt for VS 2022"**（或您安装的版本）

3. **在开发者命令提示符中运行：**
   ```bash
   cd C:\tgkz2026
   pip install TgCrypto
   ```

4. **等待安装完成**

5. **验证安装：**
   ```bash
   python -c "import TgCrypto; print('✅ 安装成功！')"
   ```

---

### 方法 2：重启计算机后使用普通 PowerShell

如果您不想使用开发者命令提示符：

1. **重启计算机**（让环境变量生效）

2. **重启后，打开 PowerShell：**
   ```powershell
   cd C:\tgkz2026
   pip install TgCrypto
   ```

---

## 🎯 推荐

**使用方法 1（开发者命令提示符）**，因为：
- ✅ 不需要重启
- ✅ 立即生效
- ✅ 最简单

---

## 📋 完整命令（复制粘贴）

在开发者命令提示符中，直接复制粘贴：

```bash
cd C:\tgkz2026
pip install TgCrypto
python -c "import TgCrypto; print('✅ 安装成功！版本:', TgCrypto.__version__)"
```
